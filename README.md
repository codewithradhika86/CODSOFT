5. 💳 Credit Card Fraud Detection

Type: Binary Classification (Imbalanced)
Goal: Detect fraudulent transactions
Dataset: Kaggle Fraud Dataset
Algorithm: Logistic Regression, XGBoost, Isolation Forest
Key Steps: Scale → SMOTE for imbalance → train → ROC-AUC
Result: ROC-AUC ~0.98 | Focus on Recall & F1, not accuracy
